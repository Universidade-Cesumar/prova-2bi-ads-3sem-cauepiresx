## Descrição

Sistema web desenvolvido para modernizar o controle de materiais utilizados nas aulas práticas do curso técnico de Enfermagem.

O sistema substitui o controle realizado por planilhas eletrônicas, permitindo um gerenciamento mais eficiente dos materiais disponíveis em estoque.

---

## Funcionalidades da Sprint 1

- Cadastro de materiais
- Consulta de materiais
- Integração com API REST (MockAPI)
- Atualização automática da tabela
- Indicador de total de materiais cadastrados
- Indicador de total de itens em estoque
- Interface responsiva

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript ES6
- Fetch API
- Async/Await
- MockAPI.io
- GitHub
